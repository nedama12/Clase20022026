Logistic Regression - Basic Concepts

Understanding classification models

What is Logistic Regression?

Logistic Regression is a supervised machine learning algorithm used for classification problems. It predicts the probability that an observation belongs to a specific class.

Regression vs Classification

Regression models predict continuous values, while classification models predict categories. Logistic regression is used for classification, not numeric prediction.

Binary Classification

Binary classification means the model predicts one of two possible outcomes, such as Yes/No, True/False, or Approved/Rejected.

Target Variable (Class Label)

The target variable is the output the model tries to predict. In logistic regression, it is usually binary (0 or 1).

Features / Predictors

Features are the input variables used by the model to make predictions, such as age, income, or experience.

Probability Estimation

Logistic regression predicts a probability between 0 and 1 that represents the likelihood of belonging to a class.

P(y = 1) = 1 / (1 + e^-z)

Decision Boundary

The decision boundary separates classes. Typically, a threshold of 0.5 is used to classify results.

Classification Threshold

If the predicted probability is greater than 0.5, the model assigns class 1; otherwise, class 0.

Real-world Example

A bank can use logistic regression to determine whether a customer should be approved for a loan based on their financial information.
