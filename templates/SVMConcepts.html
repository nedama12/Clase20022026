SVM - Basic Concepts

Understanding classification using Support Vector Machines

What is SVM?

Support Vector Machine (SVM) is a supervised machine learning algorithm used for classification tasks. It finds the optimal boundary that separates different classes in a dataset.

Hyperplane

A hyperplane is the decision boundary that separates data into classes. In 2D it is a line, in higher dimensions it becomes a plane.

Margin

The margin is the distance between the hyperplane and the closest data points. SVM tries to maximize this margin.

Support Vectors

These are the closest points to the decision boundary and define the model.

Kernel Trick

Allows SVM to work with non-linear data by transforming it into higher dimensions.

  • Linear: Simple separation
  • Polynomial: More complex relations
  • RBF: Handles non-linear data

Advantages

  • Works well in high dimensions
  • Good with complex datasets
  • Less overfitting

Limitations

  • Computationally expensive
  • Needs tuning
  • Hard to interpret

Use Cases

  • Medical diagnosis
  • Image classification
  • Spam detection
