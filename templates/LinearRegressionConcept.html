Linear Regression - Basic Concepts

Understanding the fundamentals of regression models

What is Linear Regression?

Linear regression is a supervised machine learning algorithm used to predict continuous values. It identifies the best linear relationship between independent variables and a dependent variable.

Independent and Dependent Variables

Independent variables (X) are inputs used to make predictions, while the dependent variable (Y) is the output the model predicts.

  • Independent variables: Grid position and number of laps
  • Dependent variable: Points

Training Data

Training data consists of historical records that allow the model to learn patterns between inputs and outputs.

Regression Line

The regression line represents the best fit for the data points and is used to make predictions.

y = b0 + b1x

Slope and Intercept

The slope (b1) shows how much Y changes with X, while the intercept (b0) is the value of Y when X equals zero.

Example of Application

In this project, linear regression is used to predict race points based on starting position and number of laps.
