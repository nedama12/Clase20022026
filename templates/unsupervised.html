Unsupervised Machine Learning

Understanding clustering techniques and K-Means algorithm

What is Unsupervised Learning?

Unsupervised learning is a type of machine learning where the model identifies patterns in data without labeled outputs. Unlike supervised learning, there are no predefined answers, and the model must discover hidden structures on its own.

What is Clustering?

Clustering is a technique used to group similar data points together. The goal is to ensure that elements in the same group (cluster) are more similar to each other than to those in other clusters.

K-Means Algorithm

K-Means is one of the most popular clustering algorithms. It works by selecting a number of clusters (K), assigning each data point to the nearest centroid, and iteratively updating the centroids until convergence.

Key Concepts

  • Centroids: The center of each cluster
  • Distance: Usually Euclidean distance to measure similarity
  • Iterations: Repeating the process until stable
  • Variance: Measures how compact the clusters are
