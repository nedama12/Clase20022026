Unsupervised Learning

Understanding clustering techniques and K-Means algorithm

What is Unsupervised Learning?

Unsupervised learning is a type of machine learning where the model identifies patterns in data without labeled outputs. The model discovers hidden structures on its own.

What is Clustering?

Clustering groups similar data points. Items in the same cluster are more similar to each other than to those in other clusters.

K-Means Algorithm

K-Means selects K clusters, assigns points to the nearest centroid, and updates them iteratively until convergence.

Key Concepts

  • Centroids: Center of clusters
  • Distance: Usually Euclidean
  • Iterations: Repeat until stable
  • Variance: Cluster compactness
