Logistic Regression - Basic Concepts

What is Logistic Regression?

Logistic Regression is a supervised machine learning algorithm used for classification problems. It predicts the probability that an instance belongs to a specific class.

Regression vs Classification

Regression predicts continuous values, while classification predicts categories or classes. Logistic regression is used for classification tasks.

Binary Classification

Binary classification means there are only two possible outcomes, such as Yes/No, True/False, or Approved/Rejected.

Target Variable

The target variable is the output we want to predict. In logistic regression, it is usually a binary value (0 or 1).

Features / Predictors

Features are the input variables used to make predictions, such as age, salary, or credit history.

Probability Estimation

Logistic regression outputs a probability between 0 and 1 that represents how likely an event is to occur.

Decision Boundary

The decision boundary is the threshold used to separate classes, usually 0.5.

Classification Threshold

If the predicted probability is greater than 0.5, the model predicts class 1; otherwise, class 0.

Real-world Example

A bank can use logistic regression to decide whether to approve a loan based on a customer's financial data.
