Manual K-Means Simulation

Step-by-step implementation of the K-Means algorithm using a dataset of 100 records

Dataset Description

A dataset of 100 records was created using two numerical features, allowing visualization and clustering in a 2D space.

Algorithm Process

The K-Means algorithm was simulated through three iterations:

  • Selection of initial centroids
  • Distance calculation (Euclidean)
  • Assignment to nearest centroid
  • Centroid recalculation
  • Iteration repetition

Iterations and Results

Centroids updated each iteration, refining clusters and improving grouping accuracy.

Variance Analysis

Variance decreased over time, indicating more compact and well-defined clusters.

Final Conclusion

The simulation demonstrates how K-Means iteratively improves clustering until convergence is achieved.
