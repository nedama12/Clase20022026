Manual K-Means Simulation

Step-by-step implementation of the K-Means algorithm using a dataset of 100 records

Dataset Description

A dataset of 100 records was created using two numerical features. These features represent measurable attributes (such as age and income), allowing visualization and clustering in a 2D space.

Algorithm Process

The K-Means algorithm was simulated manually through three iterations. The process followed these steps:

  • Selection of three initial centroids
  • Calculation of Euclidean distance between each point and the centroids
  • Assignment of each data point to the nearest centroid
  • Recalculation of centroids as the average of assigned points
  • Repetition of the process for three iterations

Iterations and Results

During each iteration, the centroids were updated and the clusters became more refined. Some data points changed their assigned cluster as the centroids moved toward more optimal positions.

Variance Analysis

The variance decreased across the iterations, demonstrating that the clusters became more compact. This reduction indicates that data points are getting closer to their respective centroids, improving the quality of clustering.

Final Conclusion

The manual simulation of the K-Means algorithm provided a clear understanding of how clustering works. The iterative process successfully grouped similar data points, and the decrease in variance confirmed that the model converged to an optimal solution.
